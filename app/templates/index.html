Welcome to the Room Booking Management System

This web application allows organizations to easily book, manage, and monitor meeting rooms. Admins can manage users and teams, track room utilization, and view costs — all in one place.

Designed using Flask for the backend and Bootstrap 5 for the frontend, this project demonstrates full-stack development with database integration and authentication.
